# Algorithmic Zoning Architect (AZA) - Distribution

This repository contains the official standalone distribution of the **Algorithmic Zoning Architect**.

## About AZA
AZA is an experimental exploration of Voronoi-based mass studies and capacity-constrained power diagrams for architectural space planning and interior zoning.

## How to Run
1.  Go to the [Releases](https://github.com/Ibrahim-3d/Algorithmic-Zoning-Architect-Dist/releases) page.
2.  Download the latest `AlgorithmicZoningArchitect.exe` (or the zip bundle).
3.  Run the executable on your Windows machine.
4.  *Note: No Python installation is required.*
